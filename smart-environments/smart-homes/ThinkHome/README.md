# ThinkHome

## Overview 

**ThinkHome: Smart Home Ontology for Human Activity Recognition**

A smart home ontology to drive a Smart Home System.

The ontology focuses on home control and energy efficiency. It consists of five ontologies that cover individual aspects of the proposed application.
* ActorOntology
* BuildingOntology
* EnergyResourceOntology
* ProcessOntology
* WeatherOntology
