# BOnSAI

**BOnSAI - A Smart Building Ontology for Ambient Intelligence**

**BOnSAI** is an example of an ontology that fits in more than one DISO cluster. It fits here, in the DISO `ambient-intelligence` cluster, but it also fits in the `smart-buildings` cluster within the DISO `smart-environments` cluster.

For **BOnSAI**, see `smart-buildings`.
